# Vagrantsong

### Overview

- [ ] Theme & vibe: Cuphead-esque boss battler
- [ ] Scenario-based objectives, usually restoring a Haint's Humanity
- [ ] Game end: victory or all Vagrants Westbound (defeat)
- [ ] Round: all players take a turn (in any order)
- [ ] Turn: spend 3 coins on actions, Haint turn
  - [ ] Generally more coins on a single action improves it

### Actions

- [ ] Move: gain movement points equal to move
  - [ ] Orthogonal movement
  - [ ] Cannot move through terrain
  - [ ] Cannot land on spaces occupied by Vagrant or Haint
- [ ] Rummage: draw 1 token per coin, keep 1
  - [ ] Token limit listed on Rummage icon
- [ ] Investigate: roll 1 bone per coin
  - [ ] Read initial event text before roll
- [ ] Patch Up: roll 1 bone per coin, +1 Humanity to self per success
- [ ] Bust: roll 1 bone per coin, +1 Humanity to adjacent Haint per success
- [ ] Unique skills: target, range, bones, success value
- [ ] Booming Boxcars: natural 6 -> +1 bone

### Rummage Tokens

- [ ] Discarded to Cycle Track
- [ ] Limit 1 use per turn
- [ ] Can trade with adjacent Vagrants

### Haint

- [ ] Turn: draw from bindle, resolve action, discard to Cycle Track
- [ ] Target priority: special, action-specific, candle, hurt, nearby, Westbound, recent
- [ ] Movement: moves to be in range of target
  - [ ] Can even move and count range through terrain (Vagrants cannot)
  - [ ] Will Haunt as many Vagrants as possible
- [ ] If cannot resolve action on target, then tries against next target in priority
- [ ] Cycle effects
- [ ] Haunted effects
- [ ] Haint effects
- [ ] Mood
