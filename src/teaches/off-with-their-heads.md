# Off With Their Heads!

### Overview

- [ ] Theme & vibe
- [ ] Objective: most points after 3 rounds
- [ ] Turn overview: play card simultaneously, compare cards to determine where to mark
- [ ] Round overview: play 7/9 cards, set aside remaining 2 cards for final poker hand

### Zones

- [ ] 3 zones: Meadow (High), Woods (Mid), and Keep (Low)
- [ ] Card ranking (suit, rank) determines which zone
  - [ ] 10 < J < Q < K < A
  - [ ] Queen of Hearts rotates to the next suit every turn
  - [ ] When marking, J=Q=K=10, A=1/11
- [ ] Meadow: choose mushroom matching color, mark leftmost available space
  - [ ] At game end, only complete mushrooms score
  - [ ] Center mushroom allows either color
- [ ] Woods: mark tree matching color
  - [ ] At game end, score nodes fully surrounded by marked trees
  - [ ] 4 trees allow either color
- [ ] Keep: mark space matching color (must be at start or adjacent to a marked space)
  - [ ] At game end, score marked spaces with laurels
  - [ ] Red Keep: mark any number/color in Meadow and any number/color in Woods

### Bonuses

- [ ] Teacup: mark next Teacup with a single line to indicate available
  - [ ] Can cross through available Teacup to consume: mark different zone or different color
- [ ] Biscuit: immediately mark again in a different zone (same number and color)
  - [ ] If mark was altered by Teacut, use altered mark

### Scoring

- [ ] Final poker hand
  - [ ] +5 VP for best hand
- [ ] Meadow: completed Mushrooms
- [ ] Woods: fully surrounded nodes
  - [ ] x2 if no 2 adjacent trees have same number
  - [ ] +30 VP if all 18 spaces marked
- [ ] Keep: marked spaces with laurels
  - [ ] +2 VP for Red Keep
- [ ] Tea party guests
