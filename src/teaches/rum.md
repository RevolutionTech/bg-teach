# RUM

### Overview
- [ ] Theme & vibe: pirates collecting sets of Rum bottles
- [ ] Objective: most points from Captain Cards you control
- [ ] Game end: player hits point goal (21/18/15 pts) OR Castaway Clock reaches "7"
- [ ] Tiebreaker: most Captain Cards, highest value Captain Card

### Turn: Draw OR Play a Set

#### Drawing
- [ ] Draw from the Beach (face up): replace empty slot with a face-down card from the Shipwreck
- [ ] Draw from the Shipwreck (face down): just take it

#### Playing a Set
- [ ] Play 1+ Rum Cards to form a set; must gain control of at least 1 Captain Card
- [ ] Each card is played with only ONE side counting toward the total
- [ ] To steal a Captain: set must show MORE bottles of that color than the Captain's current number
- [ ] Rotate stolen Captain Card so the new number (= bottles played) faces you
- [ ] Can steal multiple Captains in one turn
- [ ] Value caps at "8+" — card rotates to that position but cannot increase further
- [ ] Used cards mixed back into the Shipwreck

### The Beach
- [ ] Single-bottle sides of Beach cards are communal — any player can use them when proving a set
  - [ ] Just announce which singles you're claiming when playing your set
- [ ] The 2-bottle sides are NOT communal

### Rum Trio
- [ ] Play exactly 3 single bottles of the same color → steal that color's Captain AND increase its value by +2
- [ ] Can be combined with other sets on the same turn to steal multiple Captains
- [ ] Only 3 single bottles of each color exist in the whole game

### The Parrot
- [ ] Draw Parrot from Shipwreck: reveal immediately
- [ ] Discard cards from hand (not the Parrot) equal to the Castaway Clock value (2 or 3)
- [ ] Parrot goes back into the Shipwreck; Castaway Clock advances one step
- [ ] When the LAST card is taken from the Shipwreck: player with most Rum Cards discards their entire hand; play continues

### The Tide
- [ ] Triggers when all 3 Beach cards are face down
- [ ] Flip all 3 Beach cards face up immediately
- [ ] If Parrot is revealed: all players discard cards = Clock value; Clock +1; Parrot back in; refill Beach
- [ ] If a Rum Trio appears on the Beach during Tide: all 3 Beach cards go back to Shipwreck, 3 new ones refill it
