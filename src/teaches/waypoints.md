# Waypoints (Map 1: Whistling Water National Park)

### Overview
- [ ] Roll-and-write: draw routes to visit waypoints
- [ ] Objective: most points after 4 hikes
- [ ] Game end: after the 4th hike's weather track is completed

### Each Turn
- [ ] **Roll:** one player rolls a d6; advance the weather track that many spaces clockwise, circle the space reached
  - [ ] If the roll overshoots, circle the last space on that side instead
  - [ ] The circled value = movement points for ALL players this turn
- [ ] **Move or Rest:** all players decide simultaneously
  - [ ] **Move:** draw a route from your current waypoint to a new one
    - [ ] Each contour or grid line crossed = 1 MP; intersections of two lines = 2 MP
    - [ ] River: can only be crossed at a bridge (no extra MP cost)
    - [ ] Cannot pass *through* another waypoint mid-route
    - [ ] Can retrace previous routes
    - [ ] Can spend water to gain +1 MP per water used
    - [ ] You may always move fewer MPs than available, as long as you reach a waypoint
    - [ ] After moving, circle the next space on that waypoint type's track
      - [ ] Mountains: write the underlined height value instead; multiply if the space says x2/x3
  - [ ] **Rest:** +1 water instead

### Features (passed through on a route — one benefit each, first time only)
- [ ] **Lake:** circle next space on water track (gain 1 water)
- [ ] **Woodland:** circle next space on Bear, Bird, or Rabbit track

### Special Actions (unlocked by circling that symbol on a track)
- [ ] Use on any future turn (or immediately for Camera); cross out when used; can combine multiple in one turn
- [ ] **Glider:** after visiting a Mountain, immediately glide (straight line) to a non-Mountain in the same or orthogonally adjacent grid square; circle that waypoint's track
  - [ ] No features collected from the glide path (can revisit those features later)
- [ ] **Coat:** when weather gives 1–2 MPs, gain +3 MPs this turn
- [ ] **Kayak:** route can follow the river this turn, crossing up to 3 grid lines for free along it; bridges are free
- [ ] **Backpack:** wild — use as Glider, Coat, or Kayak (choose when used)
- [ ] **Camera:** immediately "visit" a waypoint in the same or adjacent grid square you haven't visited; circle its track as if visited
  - [ ] Cross off the dot next to that waypoint — it cannot be visited again
  - [ ] Arrows on a special action = must be used immediately when circled

### End of a Hike
- [ ] Triggered when the last space on the current side of the weather track is circled
- [ ] After everyone moves/rests normally, mark final location with an X
- [ ] Each player chooses **one journal entry** to score (each can only be used once across the game):
  - [ ] **1/≠○** — 1 pt per *different type* of waypoint visited this hike
  - [ ] **2/=○** — 2 pts per waypoint of *one type* visited this hike
  - [ ] **3/○○** — 3 pts per type of waypoint visited *at least twice* this hike
  - [ ] **1/□** — 1 pt per 2 different grid squares visited this hike (round down)
  - [ ] Journal counts only waypoints visited this hike (not photos, not Campsites)
  - [ ] End the hike *at a Campsite* → double your journal score; Campsites mid-hike have no effect
  - [ ] Must always choose a journal entry, even if it scores 0
- [ ] Move clockwise to the next side of the weather track; all players gain 2 water
- [ ] Next hike starts from where you ended

### Scoring (end of game)
- [ ] **Animals &amp; Trig Points:** score = highest number circled on each track
- [ ] **Mountains:** score = sum of all values written on the Mountain track
- [ ] **Lookouts &amp; Gear:** 5 pts if 4 spaces circled; 15 pts if 5 spaces circled (per track)
- [ ] **Journal Entries:** sum of all 4 journal scores
- [ ] **Goal:** points per element + 10 bonus if fully completed
  - [ ] (a) Lakes: 2 pts each, +10 for all 9
  - [ ] (b) Woodlands: 2 pts each, +10 for all 9
  - [ ] (c) Bridges: 3 pts each, +10 for all 7
  - [ ] (d) Grid Squares: 1 pt each, +10 for all 24
