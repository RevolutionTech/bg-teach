# The Fox Experiment

### Overview
- [ ] Theme: selective breeding for fox domestication
- [ ] Objective: most VPs
- [ ] Game end: 5 rounds (generations)

### Phase 1: Selection
- [ ] Turn order: Clockwise in round 1; thereafter, determined by gear positions on supply track
- [ ] Take 3 turns in order, selecting one of each item over the phase:
  - [ ] **Male Parent**: Take from kennel or draw random from deck
  - [ ] **Female Parent**: Take from kennel or draw random from deck
  - [ ] **Supply Track Position**: Gain all visible supplies (tile + board), discard tile, place gear
- [ ] **Unlock Upgrades** (optional): Spend sets of matching trait tokens to unlock mat tracks

### Phase 2: Breeding (simultaneous)
- [ ] **Steps per pup** (must complete one pup entirely before starting next):
  - [ ] Draw blank pup card
  - [ ] Gather dice: All trait dice on both parents + unlocked player friendly dice
  - [ ] Roll & arrange: Match half-symbols to form full trait symbols; halves alone do nothing
  - [ ] Friendly dice/tokens: Wild; can complete any trait or form full wild symbols
  - [ ] Record traits: Mark pup's trait tracks for each completed symbol
  - [ ] Record friendliness: Total number of completed symbols
  - [ ] Name pup
- [ ] **Player-made parents**: Provide dice based on marked symbols with dice icons

### Phase 3: Research (simultaneous)
- [ ] **Score Studies**: Assign pup(s) to studies; each pup completes 1 level of 1 study
- [ ] **Collect Trait Tokens**: Gain tokens equal to marked circles on tracks
  - [ ] If breeding multiple pups: gain tokens per trait only from the BEST fox in that trait
- [ ] **Unlock Upgrades**: Spend tokens to advance tracks:
  - [ ] More Pups: Increase pups/round (up to 4)
  - [ ] Studies: Hold more cards/increase score values
  - [ ] Friendly Dice: Increase personal die pool
  - [ ] Patrons: Place meeple on card for end-game VP and immediate bonus

### Phase 4: Administration
- [ ] Clear parents: Discard current male/female parent cards
- [ ] Update Kennel: Move this round's pups to kennel; they become next round's parents
- [ ] Friendliness Rewards: 
  - [ ] Highest friendliness pup(s) move to rewards track for VPs and bonuses
  - [ ] Players without a pup on the rewards track gain red track rewards (friendly tokens)
- [ ] Reset turn order: Based on gear positions on supply track (left to right)
- [ ] Refresh board: Replace supply tiles and refill parent rows

### Scoring
- [ ] **Studies**: Points for highest level reached on each card
- [ ] **Patrons**: Points as described on claimed cards
- [ ] **Upgrades**: Points on unlocked mat spaces/milestones
- [ ] **Friendliness**: Points on rewards track where you have meeples
- [ ] **Supplies**: 1 VP per point token/science card; 1 VP per 2 leftover trait/friendly tokens
- [ ] Tiebreaker: Closest to 1st on the supply track

### Setup
- [ ] **Starting Study**: Deal 3 study cards; choose 1 to keep, discard others
