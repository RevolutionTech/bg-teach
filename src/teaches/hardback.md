# Hardback

### Overview

- [ ] Theme & vibe
- [ ] Deck-building spelling game
- [ ] Objective: most prestige points
- [ ] Game end: first to 60 prestige triggers the final round (all players get equal turns)

### Turn

- [ ] Turn overview: spell word, discard unused, resolve benefits, purchase, cleanup, draw
- [ ] Spell a word
  - [ ] Use any number of cards from hand
  - [ ] Any card can be played facedown as a wild; provides no coins/prestige/abilities
  - [ ] May use Timeless Classics in play (yours or opponents')
  - [ ] Ghost Writer: ask for help; if you use a suggested word, the helper gets 1 Ink
- [ ] Discard unused cards
- [ ] Resolve card benefits
  - [ ] Basic Benefits (above line): always trigger if played face-up
  - [ ] Genre Benefits (below line): trigger only if another card of same genre is in word or a matching Timeless Classic is in play
  - [ ] Resolve abilities in any order
- [ ] Purchase cards and Ink
  - [ ] Buy cards from Offer Row using Coins; refill immediately
  - [ ] Buy Ink tokens for 1 Coin each
  - [ ] Flush Offer Row: once per turn, before buying, if 4+ cards cost 6¢+ OR 4+ cards share a genre
- [ ] Cleanup
  - [ ] Discard word cards to personal discard (except Timeless Classics)
  - [ ] If you used an opponent's Timeless Classic, it goes to their discard
  - [ ] Return used Ink/Remover to supply
- [ ] Draw 5 cards

### Card & Token Details

- [ ] Timeless Classics (horizontal letters)
  - [ ] Stay in play if used face-up in a word
  - [ ] Provide benefits every turn they remain in play
  - [ ] Discarded only when another player uses that letter in a word
- [ ] Ink
  - [ ] Use any time before spelling to press luck: pay 1 Ink to draw 1 card
  - [ ] Must use all Inked cards in next word; cannot make them wild
  - [ ] Failure to use an Inked card forfeits turn: discard hand and draw 5
  - [ ] 3 Ink can be spent for 1 Coin during purchase phase
- [ ] Remover
  - [ ] Spend to remove Ink from a card; card returns to hand (can be wild/discarded)
