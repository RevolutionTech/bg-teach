# Cartographers

### Overview
- [ ] Theme & vibe: map-drawing flip-and-write
- [ ] Objective: most reputation stars
- [ ] Game end: after winter scoring (4 seasons total)

### Turn Structure
- [ ] Each turn: Explore → Draw → Check (repeat until season ends)

### 1. Explore Phase
- [ ] Flip top card of explore deck, place face-up so all time values stay visible
- [ ] Ruins card: immediately flip another card on top; if that's also ruins, flip one more
  - [ ] Ruins effect applies to the *next* non-ruins card revealed
- [ ] Ambush card: resolve immediately (see Ambush); no Draw phase for players
- [ ] Rift Lands card: each player draws a 1×1 square of any terrain shown on the card

### 2. Draw Phase
- [ ] Everyone draws simultaneously on their own map
- [ ] Choose one terrain type AND one shape from the revealed card
- [ ] Shape can be rotated and/or flipped freely
- [ ] Shape cannot overlap filled spaces (mountains, wastelands, monster, filled ruins) or go off-map
- [ ] Shape does NOT need to be adjacent to anything already drawn
- [ ] If no shape fits legally, draw a 1×1 square instead
- [ ] Ruins in effect: chosen shape must overlap a ruins space (if at all possible)
  - [ ] If no shape can legally overlap a ruins space, draw a 1×1 square anywhere instead
- [ ] Coins: earn a coin if you choose a shape with a coin icon, OR if you fully surround a mountain space on all 4 sides

### 3. Check Phase
- [ ] Sum all time values of revealed explore cards
- [ ] If total ≥ season's time threshold → season ends; otherwise loop back to Explore

### End of Season Scoring
- [ ] Check the season card: it names which 2 scoring cards to evaluate this season
  - [ ] Spring: A & B · Summer: B & C · Fall: C & D · Winter: D & A
- [ ] Score each of the 2 active scoring cards per your map
- [ ] +1 star per filled coin on your coin track
- [ ] −1 star per empty space orthogonally adjacent to a monster space
  - [ ] A space adjacent to multiple monster spaces still only costs 1 star

### Ambush Cards
- [ ] Arrow on card indicates which neighbor gets the monster drawn on their map
- [ ] Pass maps, the *receiver* draws the monster shape on the map they got; fill with monster terrain
- [ ] If shape doesn't fit, draw a 1×1 monster square anywhere on that map
- [ ] Pass maps back; discard ambush card out of game
- [ ] Ambush after ruins: resolve ambush normally; ruins effect carries to the next explore card

### Preparing Next Season
- [ ] Discard the top season card
- [ ] Reshuffle ALL explore cards (including ones revealed this season)
- [ ] Add 1 new ambush card from ambush deck, shuffled in face-down
  - [ ] Unrevealed ambush cards stay in the deck AND a new one is added — can stack up!

### End of Game
- [ ] Total stars across all 4 seasons; highest total wins
- [ ] Tiebreaker: fewest stars lost to monsters
