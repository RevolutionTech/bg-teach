# Paperback

### Overview

- [ ] Theme: Novelists crafting words for Fame
- [ ] Deck-builder spelling game
- [ ] Objective: Most Fame points at end of game
- [ ] Game end: Any 2 Fame piles empty OR all Common cards taken

### Turn Flow

- [ ] **Create Word**: Form word using hand and/or top Common card
  - [ ] Fame cards are Wild (0¢)
  - [ ] Two-Letter cards count as 2 letters; order cannot be reversed
  - [ ] No proper nouns, names, or places
- [ ] **Check Length**: Count total letters (including Wilds/Two-Letter cards)
  - [ ] If length ≥ current Length Track value, gain the top Common card to discard pile
- [ ] **Resolve Abilities**: Activate abilities on cards used in the word
  - [ ] "May" is optional; all other abilities are mandatory
  - [ ] Note abilities granting extra cards for next turn
- [ ] **Score Word**: Sum top-left values of all letters in word + Common card (if used)
- [ ] **Buy Cards**: Purchase any number of Offer/Fame cards using total score
  - [ ] Cost is the bottom-left value
  - [ ] Purchased cards go to discard pile; replace alternate cards if taken
  - [ ] Unspent money is lost
- [ ] **Discard**: Move all cards in word and hand to discard pile
- [ ] **Draw**: Draw 5 cards + any bonuses from abilities
  - [ ] Shuffle discard to form new draw pile only when draw pile is empty

### Key Concepts

- [ ] **Trash**: Remove card from game (do this before buying)
- [ ] **Gain**: Acquire card directly to discard pile (usually via ability)
- [ ] **Attacks**: Remain in play until your next turn; affects all other players for one round

### Scoring

- [ ] Total Fame points on all cards in deck (hand, draw, and discard piles)
- [ ] Tiebreaker: Most Common cards
