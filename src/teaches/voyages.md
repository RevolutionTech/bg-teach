# Voyages

## Trade Winds Blow (Map 1)

### Overview
- [ ] Roll-and-write nautical exploration
- [ ] Objective: most gold
- [ ] Game end: triggered when a player earns their 3rd legendary star → one final round, then score

### Each Round (in order)
- [ ] Roll 3d6 — all players use the same results simultaneously
- [ ] Pick 2 dice to MOVE: one sets direction (windrose number), one sets distance
  - [ ] Must travel the full distance unless you hit an island, the DREAD space, or the map edge
  - [ ] Always travel in a straight line
  - [ ] Can pass through or revisit previously visited spaces
  - [ ] 2-hex islands: may depart from either hex; icon/gold applies from either hex
- [ ] Use remaining die to cross off a matching duties box
- [ ] Carry out the action of the space you ended on (not spaces passed through, not previously claimed)

### Sailors & Dice Modification
- [ ] Start with 6 sailors (already outlined on sheet)
- [ ] Exhaust (cross off) a sailor to modify any die ±1
- [ ] Heroic sailors: change a die to any result
- [ ] No limit on how many sailors/heroic sailors you spend per turn
- [ ] Dice can be modified multiple times; can't go above 6 or below 1 — but 6↔1 wraps are allowed

### Actions (space you land on)
- [ ] **Sailor**: gain a sailor
- [ ] **Relic**: make one unexhausted sailor Heroic (mark a circle inside its box)
  - [ ] Can't use if no unexhausted sailors
- [ ] **Cargo**: circle leftmost unmarked section of the matching cargo track
  - [ ] Barrels are wild (any track)
- [ ] **Settlement**: sell cargo + gain a sailor (always gain the sailor, even if selling nothing)
  - [ ] Can sell up to one of each type, OR all of one type
  - [ ] Must sell left-to-right (least valuable first)
  - [ ] Once used, cross out the settlement — it's gone for both actions permanently
- [ ] **DREAD**: exhaust 2 heroic sailors to defeat it
  - [ ] If you can't, exhaust 1 sailor/heroic sailor but don't defeat it; if no unexhausted sailors, do nothing
  - [ ] Defeated DREAD space is treated as empty

### Duties
- [ ] Cross off one box matching a die result each round
- [ ] Completing a row or column immediately awards its reward — possible to complete a row and column in one round
- [ ] If no valid box to cross off, skip the duties step

### Cargo & Selling
- [ ] 4 types: fruit, gems, spice, hide; barrel = wild
- [ ] Track position = gold value at end of game
- [ ] Unsold cargo earns 1 gold each at game end

### Ways to Earn Legendary Stars
- [ ] **Cargo**: sell your 3rd piece of one type (max 1 star total from cargo)
- [ ] **Duties**: complete the middle row of the duties track
- [ ] **DREAD**: defeat it
- [ ] **Explore**: end your turn on the legendary star ocean space
- [ ] **Recruit**: gain your 12th sailor

### Scoring
- [ ] **Islands visited**: gold value printed on the island
- [ ] **Oceans explored**: 1/4/9/16/25/36/49 gold per different ocean region traveled to or through
- [ ] **Cargo sold**: gold per track position; +1 gold per unsold cargo
- [ ] **Heroes**: 2 gold per unexhausted heroic sailor
- [ ] **Defeated DREAD**: 14 gold
- [ ] **Living Legend**: 3 gold per legendary star
