# Steam Park

### Overview

- [ ] Management of steam-powered robot amusement park
- [ ] Objective: most Danari (money)
- [ ] Game end: 6 rounds

### Roll Phase

- [ ] Simultaneous rolling: lock dice on Pig board to "save" results
- [ ] To re-roll, must roll all unlocked dice; locked dice cannot be changed or moved
- [ ] Grab turn order token once all 6 dice are locked
- [ ] Last player without token gets 3 final rolls

### Dirt Phase

- [ ] Gain 1 Dirt per hazard icon on locked dice
- [ ] Gain 1 Dirt per Visitor in Park
- [ ] Adjust total Dirt based on token:
  - [ ] 1st: -4
  - [ ] 2nd: -2
  - [ ] 3rd: 0 (4p only)
  - [ ] Last: +2

### Actions Phase

- [ ] Spend locked dice in turn order
- [ ] Each action type taken once per turn (resolve all dice for that type at once)
- [ ] **Build Rides** (Wrenches):
  - [ ] Cost: 1 die per Visitor space (1/2/3)
  - [ ] Limit: max 1 ride per size (S/M/L) per turn
  - [ ] Placement: same color rides must connect orthogonally at one space; different colors or Stands cannot touch (even corners)
- [ ] **Build Stands** (Stand icon):
  - [ ] Cost: 1 die per Stand
  - [ ] Limit: max 1 of each type per turn
  - [ ] Placement: same types must connect; different types or Rides cannot touch
- [ ] **Attract Visitors** (Person icon):
  - [ ] Spend X dice: take X Visitors of choice from pool, add to bag, then draw X from bag
  - [ ] Place matching color on Ride with space; else return to pool
- [ ] **Clean Dirt** (Broom):
  - [ ] -2 Dirt per die
- [ ] **Play Bonus Cards** (Card icon):
  - [ ] Play 1 card per die; gain Danari shown
  - [ ] No duplicate cards in same turn
- [ ] **Expand Park** (Any non-blank icon):
  - [ ] Cost: 1 die per ground tile; max 2 per turn

### Income Phase

- [ ] Gain 3 Danari per Visitor in Park
- [ ] Hand limit: 3 cards
- [ ] Draw back to limit: draw 2, keep 1 until at 3
- [ ] Reset turn order tokens and advance turn counter

### Stand Abilities

- [ ] Security: Allows mulligan 1 Visitor from bag
- [ ] Info Point: Allows placement of 1 Visitor of wrong color (Visitor expires EOT)
- [ ] Casino: Change 1 die
- [ ] Toilet: Double 1 clean die (can't double count dice)
- [ ] Promotion: Double 1 Visitor die (can't double count dice)

### Scoring

- [ ] Pay Danari fee for total Dirt based on Final Dirt Track
- [ ] 30+ Dirt: automatic loss (0 money)
- [ ] Tiebreaker: most Visitors

### Setup

- [ ] Choose 3 Bonus cards from 6 starting cards
