# Hearts

### Overview

- [ ] Summary: trick avoidance game with shoot the moon
  - [ ] Must-follow, no trumps
- [ ] Objective: fewest points
- [ ] Game end: first player hits 50 points
- [ ] Deck composition: no 2♦️ in 3/5 player games, no 2♣️ in 5 player games

### Hand

- [ ] Each ♥️ = 1 point
- [ ] Q♠️ = 13 points
- [ ] Shoot the moon: win all ♥️ and the Q♠️ to score 0, other players score 26 points
- [ ] 2♣️ / 3♣️ leads with that card
  - [ ] Cannot play ♥️/Q♠️ off-suit on this trick
  - [ ] ♥️ cannot be led until broken
