# Gutenberg

### Overview

- [ ] Theme & vibe
- [ ] Objective: most fame points
- [ ] Game end: 6 rounds
- [ ] Round overview: Plan, Execute Plans, Fulfill Orders, Prepare Next Round

### Resources

- [ ] Consumable: ink, guilders
- [ ] Non-consumable: types, specialty levels

### Plan

- [ ] Simultaneously and secretly allocate initiative markers to 5 action rows behind screen
- [ ] Max 6 markers per row
- [ ] Reveal screens and resolve actions top to bottom
- [ ] Action player order: most markers to fewest; ties broken by current turn order
- [ ] Must place >= 1 marker to perform action; performing action is optional

### Execute Plans

- [ ] **Take Orders**: take 1 printing card and 1 refinement card from board to form order pair
  - [ ] Permanently linked; max 4 orders in storage (can discard to free up space)
- [ ] **Take Inks**: select set of 3 inks on board, take 1–3 inks from left to right
  - [ ] Leftmost is free, center costs 1 guilder, rightmost costs 2 guilders
  - [ ] Position costs remain fixed even if set was partially emptied by previous player
- [ ] **Develop Specialties**: take 1 specialty card from board
  - [ ] Advance indicated specialty track by 1 level per icon, OR advance any other single track by 1 level
  - [ ] Reaching level II/IV/V/VI for first time advances reward track and grants a reward (ink, order, or type)
  - [ ] Level VI is max; further track advances grant 3 guilders instead
- [ ] **Improve Printing Houses**: take 1 gear from board and install on printing house board cylinders
  - [ ] Max 3 gears; must start at top and mesh successively; exactly 1 sector must align with active mark
  - [ ] Alternatively: uninstall, rotate, and reinstall 1 gear
  - [ ] Used gears this round cannot be moved or discarded
  - [ ] Claim active sector reward once per round per gear by placing a used gear marker (max 3 markers per round)
- [ ] **Patronage**: claim 1 available reward (blocks others) OR take 1 patronage card
  - [ ] Rewards: take 1 order, advance 1 specialty 1 level, take 3 guilders, or take 2 inks from bag
  - [ ] Patronage cards: must meet specialty, ink, and/or type requirements; must be below/left of round marker (unavailable rounds 1–2)

### Fulfill Orders

- [ ] Clockwise order from first player
- [ ] Allocate resources to orders; then fulfill orders simultaneously
- [ ] Fulfilling printing card is mandatory
- [ ] Fulfilling refinements is optional (two independent parts):
  - [ ] Both parts: additional reward
- [ ] Unfulfilled orders remain in storage

### Rewards

- [ ] Order: draw 2 printing and 2 refinement, keep 1 of each; can pay 2 guilders to draw 2 additional cards repeatedly

### Buying Types

- [ ] Non-action; buy at any time from unlimited supply
- [ ] Cost of n-th type is n guilders (e.g., 4th costs 4, 5th costs 5)

### Prepare Next Round

- [ ] Reset board
- [ ] Pass first player token clockwise
- [ ] Every player gives 1 initiative marker to player who just passed token
  - [ ] 1st/2nd/3rd/4th player should have 7/8/9/10 initiative markers
- [ ] Rotate top gear clockwise 1 sector; connected gears also turn

### Scoring

- [ ] Fame track points scored during game
- [ ] Specialties: 6/3/1 points for level VI/V/IV
- [ ] Patronage cards: 8 points each
- [ ] Guilders: 1 point per 3 guilders
- [ ] Tiebreakers: fewest types, then fewest inks

### Setup

- [ ] Draw 2 random character tiles, choose 1 to keep
- [ ] Snake draft starting orders from face-up pool of 6/8/10 pairs
- [ ] Choose 3 starting types from pool
