# Robinson Crusoe: Adventures on the Cursed Island

### Overview
- [ ] Theme &amp; vibe: brutal cooperative survival — you're castaways fighting nature, weather, wounds, and bad luck
- [ ] Objective: complete scenario goal before time runs out
- [ ] Game end: win immediately on fulfilling scenario goal; lose immediately if any character dies or the final round ends without winning
- [ ] Everything is shared info — no secrets; all cards drawn openly

### Characters
- [ ] Each player has 2 Action pawns, a Wound track, 4 special abilities (cost Determination tokens to activate)
- [ ] Each ability can only be used once per round
  - [ ] Abilities can be used at any time unless stated otherwise
- [ ] Each character has a personal Invention only they can build
- [ ] Wounds: move marker right for each wound; skip morale arrows lower group morale by 1
  - [ ] Reach the skull = instant death = everyone loses

### Round Structure (6 phases)

**1. Event Phase** *(skip round 1)*
- [ ] First Player draws top of Event deck
  - [ ] If Adventure or Mystery card: apply its Event Effect, discard, draw again
- [ ] On an Event card: resolve book symbol effect (from Scenario sheet); place Adventure token if shown; apply Event Effect
- [ ] Slide card into right Threat space — leftmost card pushed off triggers its Threat Effect immediately
- [ ] Threats can be removed via the Threat Action before they're pushed off

**2. Morale Phase**
- [ ] First Player gains or loses Determination tokens based on morale track position (−3 to +2; rightmost = +2 OR heal 1 wound)
- [ ] If can't discard enough tokens: suffer 1 wound per missing token
  - [ ] Determination tokens are personal — cannot be traded

**3. Production Phase**
- [ ] Gain 1 resource per source on the Island tile containing camp
- [ ] Resources go to Available Resources space (immediately usable)

**4. Action Phase** *(the main event)*
- [ ] Planning first: all pawns placed simultaneously before any actions resolve
  - [ ] All requirements (resources, Items, Weapon level) must be met at planning time — not "we'll have it after this action"
  - [ ] Resources assigned to an action are locked; can't be used elsewhere
- [ ] 2 pawns → automatic success; 1 pawn → roll 3 dice (Wound / Success / Adventure), all independent
  - [ ] Failure on Success die returns resources and gives 2 Determination tokens
- [ ] Stacked pawns: top pawn resolves the action (gains rewards/suffers wounds); others support only
- [ ] Side-by-side pawns: resolve the same action twice
- [ ] Resources/Inventions/Discovery tokens gained during actions go to **Future Resources** space — unavailable until end of Action Phase
  - [ ] Exceptions: Determination tokens, wounds, weapon/palisade/roof level changes apply immediately

**Actions:**
- [ ] **Threat**: place 1–2 pawns on Event card to gain benefit and discard it (no dice)
- [ ] **Hunt**: exactly 2 pawns on Hunt space; draws Beast card; compare Weapon level to Beast strength (difference = wounds); lowers Weapon level; gains food/fur in Future Resources
- [ ] **Build**: 1–2 pawns + resources on an Invention card, or on Shelter/Roof/Palisade/Weapon track
  - [ ] Requires terrain type to be explored (black marker must cover the icon on the Invention)
  - [ ] Each Invention can only be attempted once per round
  - [ ] Built Inventions land in Future Resources; flip to Item side and activate at end of Action Phase
- [ ] **Gather**: 1–2 pawns above a resource source on an adjacent tile; each source once per round; not on camp tile
  - [ ] +1 pawn per Island tile of distance from camp
- [ ] **Explore**: 1–2 pawns on an unexplored space adjacent to explored tiles; reveals Island tile
  - [ ] Check terrain type → unlock Inventions; check for beast symbol → add to Hunting deck; check totem → scenario effect; draw Discovery tokens
  - [ ] +1 pawn per tile of distance from camp
- [ ] **Arrange Camp**: 1 pawn → +2 Determination tokens and +1 morale (4p: choose one)
- [ ] **Rest**: 1 pawn → heal 1 wound

**Neutral Action Pawns:**
- [ ] Color-coded: red = Hunt, brown = Build, grey = Gather, green = Explore
- [ ] Can only be used as support (never resolves an action alone)

**End of Action Phase:**
1. Move Future Resources to Available Resources
2. Activate Inventions → flip to Item side → place on board
3. Apply treasure effects

**5. Weather Phase**
- [ ] Roll dice shown on Scenario sheet for current round
- [ ] Snow clouds: discard 1 wood each (Roof irrelevant — it's cold)
- [ ] All clouds (snow + rain): subtract Roof level; pay 1 food + 1 wood per unprotected cloud
- [ ] Hungry Animals die: discard food, lower Palisade, or fight strength-3 Beast (each player suffers wounds if unmet)
- [ ] Storm token: lower Palisade by 1 (wound if impossible)

**6. Night Phase**
- [ ] Each character eats 1 food — 2 wounds if unable
- [ ] Optional: move camp to adjacent tile
  - [ ] Shelter kept; halve Roof and Palisade (round up)
  - [ ] No Shelter = Roof/Palisade drop to 0
- [ ] No Shelter (and no natural shelter) = 1 wound per character
- [ ] Discard perishable food that can't be stored
- [ ] Healing happens here via Items/Treasures/Discovery tokens
- [ ] Remove markers from special abilities; advance Round marker; pass First Player token

### Key Rules to Highlight
- [ ] **Unfulfilled Demand**: if you can't pay a cost, you suffer wounds — you cannot choose wounds instead of paying if you *can* pay
- [ ] **Future vs Available Resources**: anything from an action goes to Future; anything from outside the Action Phase goes to Available
- [ ] **Token exclusivity**: only 1 token of each type per space; failing to place a *marker* causes wounds; failing to place a *token* does not
- [ ] **Golden rule**: Scenario sheet overrides rules; card text overrides Scenario sheet

### Mystery Cards
- [ ] Drawn when a card/token instructs it; specify type (treasure / creature / trap) and maximum count
- [ ] Reveal one at a time until you hit an allowed type; others set aside
- [ ] After each resolved card, decide to keep drawing or stop
- [ ] Traps/creatures: immediate; treasures: Future Resources (effect at end of Action Phase)
- [ ] Set-aside cards shuffled back into deck at end

### Discovery Tokens
- [ ] Found during Explore; land in Future Resources; available at end of Action Phase
- [ ] Fallen Tree and Nourishing Larvae must be cashed in immediately when available
- [ ] Others can be held and used at almost any point

### Combat
- [ ] Weapon level vs. Beast strength → difference = wounds to resolve
- [ ] Can use Treasures/Starting Equipment/abilities to temporarily boost Weapon level before comparing (track doesn't move)
- [ ] Weapon level is then reduced by the value on the card
- [ ] Palisade reduced if card calls for it; wounds if Palisade already 0
