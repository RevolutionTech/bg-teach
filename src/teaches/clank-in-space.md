# Clank! In! Space!

### Overview

- [ ] Theme & vibe
- [ ] Objective: steal an Artifact and escape with the most points
- [ ] Game End: all players escape or are knocked out

### Flow

- [ ] Turn overview: play all cards from hand in any order, resolve effects, buy cards, move, then discard everything and draw back up to 5

### Concepts

- [ ] Resources:
  - [ ] Skill (Blue): buy new cards from the Adventure Row
  - [ ] Swords (Orange): fight enemies in the Adventure Row or on the board
  - [ ] Boots (Yellow): move between spaces
- [ ] Faction bonuses: resolve only if you played another card of that faction this turn
- [ ] Clank!: place your color cubes into the Clank! area
- [ ] Data cubes: used to hack terminals

### Movement & Board

- [ ] Movement: 1 Boot per border crossed
- [ ] Locked Borders: requires a Master Key (from Market)
- [ ] Enemy Borders: crossing a border with a monster costs 1 Health per icon (or spend 1 Sword per icon to ignore)
- [ ] Telepads: move to any other Telepad (requires TelePass market item), depletes Boots that turn
- [ ] Hyperlifts: move to any adjacent Hyperlift station, depletes Boots that turn
- [ ] Terminals: place a Data Cube
  - [ ] Must hack 2 Terminals in different modules to enter the Command Center
- [ ] Market: spend 7 Credits to buy: Master Key, TelePass, Med Kit (immediately heal 2), or Contraband (required for some card effects)
  - [ ] Cannot buy same item twice (include Med Kit)

### Main Actions

- [ ] Play cards
- [ ] Buy cards: use Skill to buy from Adventure Row or the reserve (FAZR, Boldly Go, Memory Core)
- [ ] Fight: use Swords to defeat Monsters in the Row or the G0B-L1N
- [ ] Market: if on a Market space, spend Credits for items

### Lord Eradikus Attacks

- [ ] Trigger: pulling a card with the Red Dragon icon from the deck when refilling Adventure Row
- [ ] Resolution:
  - [ ] Put all cubes from Clank! Area into the Bag
  - [ ] Pull N cubes based on Rage Track
  - [ ] Colored cubes go to health tracks; black cubes are set aside
- [ ] Rage Track: advance when a command code is completed, an Artifact is picked up, or an archive is discovered
  - [ ] Bounty Hunters: cube that attacks all players when drawn
  - [ ] Blockade: severs connection between Hyperlift and Cargo Bay

### Game End & Escape

- [ ] Getting an Artifact: move to an Artifact space (must have 2 hacked terminals)
  - [ ] Cannot swap the Artifact later
- [ ] The Escape: once you have an Artifact, return to the Cargo Bay
- [ ] Triggering the End: when a player escapes or dies, on their turn they pull 6/4/4 cubes from the Bag

### Scoring

- [ ] Artifact value
- [ ] Points on cards in deck
- [ ] Acquired tokens
- [ ] 20 Point bonus if you escaped (reached Cargo Bay)
- [ ] 0 Points if you die in the "Red" (Command Center/Modules)
- [ ] Tiebreaker: most valuable Artifact

### Miscellaneous

- [ ] Credits do not count as points, used only for buying at Markets
- [ ] Power Crystals: Some cards/effects require you to have a Crystal in your possession to trigger
